# Description:
Live Chat Application with Python, WebSocket and SQLite Database.
Simple chat application. Enter your username, select connected user, view chat history, and send messages.


# Tech Stack:
- Python + FastAPI
- WebSocket
- JavaScript
- SQLite
- HTML/CSS
